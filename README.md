# React_Websocket_sensor_dashboard
Sensor view dashboard using React Hooks and Websocket API

#How to Install and Run the Project
> Run `npm install` in `server` folder.
>Navigate to `server` folder. Here run `npm start`
>Run `npm install` in root folder.
>In the root folder run `npm start`

After this you will be able to see all the connected and disconnected websockets on `http://localhost:3000/`.

#Tech Stack used
>React.js
>ReactStrap
>WS npm library for the websocket `https://www.npmjs.com/package/ws`
